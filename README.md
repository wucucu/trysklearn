超级简陋的 scikit learn 入门简介
----
# 预备动作

## python3 和 git
```sh
> git clone https://github.com/wucucu/trysklearn.git
> cd trysklearn
```

## 装第三方的python packages
```sh
> python3 -m venv .env
> source .env/bin/activate
> pip install --upgrade pip
> pip install -r requirements.txt
```

## jupyter notebook
```sh
> cd notebooks
> jupyter notebook
```
